### Homework Description
The goal is to create and deploy a responsive personal web presentation/application that showcases your portfolio, resume, and academic projects. The project must be built using modern web standards (HTML5, CSS3, ES6+ JavaScript) and can optionally utilize CSS frameworks like Tailwind or Bootstrap.

#### Structural & Design Requirements
* **Pages:** The website must consist of a minimum of 3 distinct HTML pages, specifically covering:
  1. **Home Page (Úvodní stránka):** An introductory landing page.
  2. **Personal Profile & Resume (Životopis):** A professional profile and CV overview.
  3. **First Semester Projects (Projekty):** A dedicated section displaying projects completed during your first semester.
* **Responsive Design:** The layout must be fully responsive, ensuring seamless functionality and visual presentation across desktop, tablet, and mobile displays.

#### Technical Implementation
* **Languages:** Semantic HTML5 markup, customized CSS3 layout rules (or utility classes via Tailwind/Bootstrap), and modern JavaScript (ES6+).
* **Interactivity Component:** At least one native JavaScript interactive feature must be integrated into the application. Examples include:
  * A light/dark theme toggle switcher.
  * Client-side form validation.
  * An image slider/carousel.
  * An embedded interactive map.
  * A simple asynchronous third-party API fetch call.
