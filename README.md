# 460-2079/01 – Fundamentals of Information Technology (ZIT)

## Course Objectives Expressed as Achieved Skills and Competencies
The course aims to provide students with a fundamental overview of technologies and their practical application for subsequent stages of their studies. The objective is for students to gain awareness of specific tools and technologies they will encounter in following courses. This will ensure they are prepared to use these technologies effectively and further expand their knowledge within specialized subjects. At the same time, they will gain insight into how these topics interconnect within the broader field of IT.

## Course Annotation
Students will gain an overview of various technologies and areas that cover current modern trends in IT, with an emphasis on their subsequent practical application. The course serves as a foundation for understanding areas that students might not have encountered in their previous studies, or for integrating these areas into the overall concept of the Computer Science study program.

## Additional Student Requirements
The student should be able to navigate the concepts and topics covered in class. They should be able to create a web presentation on a required topic.

## Prerequisites
The course has no prerequisites.

## Corequisites
The course has no corequisites.

## Course Syllabus

### 1. Fundamental Concepts, Tools, and Workflows
- Basic concepts such as domain, DNS, IP address, port, HTTP protocol
- Command line operations (Linux, PowerShell)
- Work and development environments (VS Code, etc.)

### 2. Introduction to Version Control with GIT
- GIT vs. GitHub
- Repository creation and management, basic Git operations (commit, push, pull)
- Working with branches, conflict resolution, principles of team collaboration

### 3. Introduction to Containerization with Docker
- Working with basic Docker commands: creation, management, and running of containers
- Basics of Dockerfile creation and container management on a local machine
- Orchestration

### 4. Basics of Web Development: HTML and CSS
- Structure of an HTML (HTML 5) document, basic elements, and attributes
- CSS: selectors, box model, basic principles of responsive design

### 5. Introduction to Using JavaScript on the Web
- Basic JavaScript syntax, DOM manipulation
- Events
- HTML 5 API
- JSON and XML

### 6. AI and the Use of Artificial Intelligence Tools
- Fundamental AI concepts and trends in AI application
- Chatbots, text and image generation, data analysis
- Use of AI in education, ethics, and formalities

### 7. Fundamentals of IT Security and Cyber Hygiene
- Basic overview of security threats
- Personal data protection and secure computer usage, including social networks
- Principles of safe usage of modern technologies
